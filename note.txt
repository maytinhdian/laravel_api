Các cách test API: 
    - Phần mềm PostMan 
    - Extension Visual Studio Code 
    - Dịch vụ test online  
Xây dựng CRUD API: 
    - GET: /users => Lấy tất cả tài nguyên 
    - GET: /users/id => Lấy 1 tài nguyên theo ID 
    - POST: /users => Thêm mới tài nguyên 
    - PUT: /users/id => Cập nhật tất cả các trường của 1 tài nguyên 
    - PATCH: /users/id => Cập nhật 1 vài trường của 1 tài nguyên 
    - DELETE: /users/id => Xóa 1 tài nguyên
